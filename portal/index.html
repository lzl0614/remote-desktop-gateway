Quick Start

  1. Open the Guacamole login page.
  2. Enter the account issued for this gateway.
  3. Pick your device connection, such as `PC-1`.

Current Stack

  • Browser access through an HTTPS gateway
  • Guacamole manages all remote connections
  • Windows devices join through reverse SSH tunnels

Security Note

Keep public `3389` closed. Device onboarding, account issuance, and password rotation should all stay inside a controlled workflow.
